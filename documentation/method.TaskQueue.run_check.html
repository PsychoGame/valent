Description [src]

A variant of valent_task_queue_run() that checks if a task succeeds.

If task reports an error, any queued tasks will be cancelled and no new tasks will be accepted by queue.

Be aware that tasks are executed in order of priority, so tasks queued before task may be affected if it has a higher priority.
