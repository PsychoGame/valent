Description

ValentSession is an aggregator of desktop sessions, with a simple API generally intended to be used by ValentDevicePlugin implementations.

Plugins can provide adapters for session selections by subclassing the ValentSeesionAdapter base class. The priority of session adapters is determined by the .plugin file key X-SessionAdapterPriority.
