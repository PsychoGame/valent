ValentMedia is an aggregator of desktop media players, with a simple API generally intended to be used by ValentDevicePlugin implementations.

Plugins can provide adapters for media players by subclassing the ValentMediaPlayerProvider base class to register ValentMediaPlayer instances by emitting ValentMediaPlayerProvider::player-added.
