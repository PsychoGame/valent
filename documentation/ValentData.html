Description

The ValentData class is intended to abstract storing and loading data for a given context, such as a device or plugin.

There are conveniences for working with GFile, getting and settings file contents and high-level sqlite.

The “context” property

  “context”                  char *

The "context" property is the specific context for this ValentData.

For example, the application will generally have no context (NULL), therefore using the root of “base-path” , while devices or backends will generally store their data in subdirectories of these.

Owner: ValentData

Flags: Read / Write / Construct Only

The “parent” property

  “parent”                   ValentData *

The "parent" ValentData of this instance. If given during construction the ancestor ValentData will be taken into consideration when using path and filenames.

For example, a ValentManager, with a ValentData object, manages devices with ValentData objects that are children of the manager's data object, and so on with plugins.

Owner: ValentData
