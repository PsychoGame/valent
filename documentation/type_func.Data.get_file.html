Description [src]

Create a new GFile for filename in base_path. If unique is TRUE, the file will be guaranteed not to conflict with an existing file.

Parameters

dirname const char*
 

A directory path.
 The data is owned by the caller of the function.
 The string is a NUL terminated UTF-8 string.
basename const char*
 

A file name.
 The data is owned by the caller of the function.
 The string is a file system path, using the OS encoding.
unique gboolean
 

Whether the resulting filepath should be unique.

Return value

Returns: GFile
 

A GFile.
