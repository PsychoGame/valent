Description

ValentInput is an abstraction of input controllers, with a simple API generally intended to be used by ValentDevicePlugin implementations.

Plugins can provide adapters for input controllers by subclassing the ValentInputController base class. The priority of input controllers is determined by the .plugin file key X-InputControllerPriority.
