Class

ValentClipboardAdapter

Description [src] 

abstract class Valent.ClipboardAdapter : GObject.Object {
  parent_instance: GObject
}

ValentClipboardAdapter is a base class for plugins that provide read-write access to the desktop clipboard for use by remote devices.

.plugin File

Plugins require no special entries in the .plugin file, but may specify the X-ClipboardAdapterPriority field with an integer value. The implementation with the lowest value will take precedence.
