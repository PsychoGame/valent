Class

ValentInputAdapter

Description [src] 

abstract class Valent.InputAdapter : GObject.Object {
  parent_instance: GObject
}

ValentInputAdapter is a base class for plugins that provide a means to simulate keyboard and pointer events on the host system on behalf of remote devices.

.plugin File

Plugins require no special entries in the .plugin file, but may specify the X-InputAdapterPriority field with an integer value. The implementation with the lowest value will take precedence.
