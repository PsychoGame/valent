Class

ValentData

Description [src] 

class Valent.Data : GObject.Object {
  parent_instance: GObject
}

The ValentData class is intended to abstract storing and loading data for a given context, such as a device or plugin.

There are conveniences for working with GFile, getting and settings file contents and high-level sqlite.
