Description

ValentPower is an abstraction of desktop power sources, with a simple API generally intended to be used by ValentDevicePlugin implementations.

The ValentPower singleton mirrors the default ValentPowerDevice from the default ValentPowerDeviceProvider, so should generally be used directly.
