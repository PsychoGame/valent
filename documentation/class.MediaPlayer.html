Class

ValentMediaPlayer

Description [src] 

class Valent.MediaPlayer : GObject.Object {
  parent_instance: GObject
}

A ValentMediaPlayer is a base class for media players that more or less mirrors the MPRISv2 specification. The primary difference is that media player control is not spread across several interfaces as is the case with the application, player, playlist and tracklist interfaces described by MPRIS.

The Valent MPRIS plugin already provides support for MPRISv2 compatible players exposed on DBus, so it is primarily made available to allow other media players to be supported by plugins.

Properties

Valent.MediaPlayer:flags

A bitmask of ValentMediaActions that are actionable on the player.

Valent.MediaPlayer:metadata

The metadata of the current element.

Valent.MediaPlayer:name

The name of the media player, corresponding to the Identity property of the org.mpris.MediaPlayer2 interface.

Valent.MediaPlayer:position

The current track position in microseconds, between 0 and the ‘mpris:length’ metadata entry (see ValentMediaPlayer:position).

Valent.MediaPlayer:state

A bitmask of ValentMediaState that describes the playback state.

Valent.MediaPlayer:volume

The volume level.

Signals

Valent.MediaPlayer::changed

ValentMediaPlayer::changed is a convenience function for notifying of multiple property changes. Implementations may emit GObject::notify but must emit ValentMediaPlayer::changed.

Valent.MediaPlayer::seeked

ValentMediaPlayer::seeked is emitted when the track position has changed in a way that is inconsistant with the current playing state.
