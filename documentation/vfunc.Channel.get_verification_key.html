Get a verification key for the connection.

Implementations should return a string for the user to confirm the connection is being made by a particular device, similar to a Bluetooth PIN.

Return value

Returns: const char*
 

A verification key.
 The data is owned by the called function.
 The string is a NUL terminated UTF-8 string.
