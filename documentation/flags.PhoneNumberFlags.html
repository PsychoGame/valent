Enumeration

ValentPhoneNumberFlags

Declaration 

flags Valent.PhoneNumberFlags

Description [src]

Flags that describe a ValentPhoneNumber type, capabilites or other attributes such as whether the number is the preferred or primary number.

These values are based on RFC2426 (vCard 3.0), however only HOME, CELL, WORK and PREF are used in Valent.
