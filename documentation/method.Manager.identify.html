Description [src]

Request a connection from the device at uri if given, otherwise ask each loaded ValentChannelService to identity itself on its respective network.

The uri argument is string in the form backend://address, where “backend” is the module name and “address” is some address, DBus path or serialized data the backend can use to contact the device. A complete URI for the Lan backend might look like lan://192.168.0.10:1716.

Parameters

uri const char*
 

A URI.
 The data is owned by the caller of the function.
 The string is a NUL terminated UTF-8 string.
