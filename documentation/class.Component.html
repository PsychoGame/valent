Class

ValentComponent

Description [src] 

abstract class Valent.Component : GObject.Object {
  parent_instance: GObject
}

ValentComponent is a base class for session and system components, such as the clipboard or volume control. Each ValentComponent is typically a singleton representing a unique resource, but may be backed by one or more extensions implemented by PeasExtensions.

Instance methods

valent_component_get_extensions

Get a list of the currently loaded extensions.

valent_component_get_priority_provider

Get the extension with the highest priority for component. The default value for extensions is 0; the lower the value the higher the priority.

Properties

Valent.Component:plugin-context

The context for the component. This is a GSettings safe string such as “contacts” or “media”, used to distinguish the settings and configuration files of pluggable extensions.

Valent.Component:plugin-type

The GType of the extension this component aggregates.

Signals

Valent.Component::extension-added

The “extension-added” signal is emitted when component has enabled a supported extension.

Valent.Component::extension-removed

The “extension-removed” signal is emitted when a extension has discovered a extension is no longer available.
