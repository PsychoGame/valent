Prerequisites

ValentDeviceActivity requires GObject.

Description

The ValentDeviceActivity interface is typically implemented by device plugins that need a large widget to display controls.

A consumer of ValentDeviceActivity widgets should typically present these as pages in a GtkStack or the content of a GtkDialog.
