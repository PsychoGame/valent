Prerequisites

ValentDeviceGadget requires GObject.

Description

The ValentDeviceGadget interface is typically implemented by device plugins that need a small widget to display controls.

A consumer of ValentDeviceGadget widgets should typically present these as items in an action bar or toolbar.
