Description [src]

ValentManager::device-added is emitted when a new ValentDevice has been added to manager. This usually happens when a ValentChannelService emits ValentChannelService::channel with an unknown identity.

Note that the internal state of manager has already been updated when this signal is emitted and manager will hold a reference to device.

Default handler:

The default handler is called after the handlers added via g_signal_connect().

Parameters

device ValentDevice
 

A ValentDevice.
 The data is owned by the caller of the function.
