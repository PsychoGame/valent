Class

ValentChannel

Description [src] 

class Valent.Channel : GObject.Object {
  parent_instance: GObject
}

The ValentChannel object is a base class for implementations of device connections in Valent. Typically these are created by a ValentChannelService implementation that wraps a GIOStream connection before emitting ValentChannelService::channel.

Implementations will usually implement at least valent_channel_download() and valent_channel_upload() for data transfer between devices. They may also implement valent_channel_store_data() for storing channel specific data when a devices is paired (eg. a TLS certificate).

Gets the GIOStream for the channel, or NULL if unset.

valent_channel_get_identity

Gets the identity packet sent by the ValentChannelService during connection negotiation.

valent_channel_get_peer_identity

Gets the identity packet sent by the peer during connection negotiation.

valent_channel_get_uri

Return a string representation of channel that can be used to reconnect to the device, by passing to valent_channel_service_identify().

valent_channel_get_verification_key

Get a verification key for the connection.

valent_channel_read_packet

Asynchronously read the next JsonNode packet from channel. Call valent_channel_read_packet_finish() to get the result.

valent_channel_read_packet_finish

Finishes an operation started by valent_channel_read_packet().

valent_channel_set_uri

Set the string representation of channel that can be used to reconnect to the device, by passing to valent_channel_service_identify().

valent_channel_store_data

This is called when a device is paired, allowing implementations to store data required to authenticate during later connections.

valent_channel_upload

Offer a connection to the remote device. Implementations are expected to populate packet with information to negotiate the connection (eg. port).

valent_channel_write_packet

Asynchronously write the JsonNode packet to channel. Call valent_channel_write_packet_finish() to get the result.

valent_channel_write_packet_finish

Finishes an async operation started by valent_channel_write_packet().

Properties

Valent.Channel:base-stream

The base GIOStream for the channel.

Valent.Channel:identity

The identity packet sent by the ValentChannelService. In other words, this identity packet represents the local device.

Valent.Channel:peer-identity

The identity packet sent by the peer. In other words, this identity packet represents the remote device.

Valent.Channel:uri

The reconnect URI of the channel.
