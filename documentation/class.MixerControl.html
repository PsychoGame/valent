Class

ValentMixerControl

Description [src] 

abstract class Valent.MixerControl : GObject.Object {
  parent_instance: GObject
}

ValentMixerControl is an base class for audio mixer controls.

Ancestors

  • GObject

Instance methods

valent_mixer_control_emit_stream_added

Emit the ValentMixerControl::stream-added signal. This should only be called by implementations after stream has been added.

valent_mixer_control_emit_stream_changed

Emit the ValentMixerControl::stream-changed signal. This should only be called by implementations after stream has changed.

valent_mixer_control_emit_stream_removed

Emit the ValentMixerControl::stream-removed signal. This should only be called by implementations after stream has been removed.

Properties

Valent.MixerControl:plugin-info

The PeasPluginInfo describing this mixer control.

Signals

Valent.MixerControl::stream-added

The “stream-added” signal is emitted when a new stream is added to the ValentMixerControl.

Valent.MixerControl::stream-changed

The “stream-changed” signal is emitted when a stream is changed from the ValentMixerControl.

Valent.MixerControl::stream-removed

The “stream-removed” signal is emitted when a new stream is removed from the ValentMixerControl.
