Description

ValentClipboard is an aggregator of desktop clipboard sources, with a simple API generally intended to be used by ValentDevicePlugin implementations.

Plugins can provide adapters for clipboard selections by subclassing the ValentClipboardSource base class. The priority of clipboard sources is determined by the .plugin file key X-ClipboardSourcePriority.
